# 基于大模型与多智能体的电力工程设计评审系统 — 融合方案

## 一、系统概述

本系统面向电力设计院（输变电、新能源、配网工程），通过 **7个专业Agent + 1个主审Agent** 的协同架构，实现从图纸上传、规范合规检查、造价分析到结构化报告生成的全流程智能化评审。融合方案既保留了产品化的交互闭环，又注入了电力工程的专业深度，预计可将评审周期从传统 **5-7天缩短至1-2天**，重大缺陷漏检率显著降低。

## 三、融合后的智能体设计（7+1架构）

| 智能体 | 来源融合 | 核心能力 | 关键技术 |
|---|---|---|---|
| **电气一次Agent** | 我方案 | 主接线可靠性分析、短路电流校验、设备选型匹配 | 领域大模型推理 + 短路电流计算API |
| **电气二次Agent** | 我方案 | 继电保护配置逻辑、自动化系统合规性、通信点表 | 保护整定规则引擎 + LLM推理 |
| **土建Agent** | 我方案 | 结构荷载计算、地基处理、抗震设防、阴影遮挡分析 | 结构计算API + 太阳高度角算法 |
| **线路Agent** | 我方案 | 路径优化、杆塔排位、导线力学、电缆载流量校验 | 力学计算 + 路径规划算法 |
| **规范合规Agent** | 上传方案 | 跨规范检索、强制性条文比对、引用溯源 | RAG检索增强 + 向量数据库 + 混合检索 |
| **造价分析Agent** | 上传方案 | 工程量清单提取、历史造价对比、异常单价检测 | NLP抽取 + 孤立森林 + 工程量联动校验 |
| **安全评估Agent** | 上传方案 | 电气安全距离、接地设计、消防通道、防孤岛保护 | 规则引擎 + LLM推理 + 风险知识图谱 |
| **主审Agent** | 两者融合 | 汇总7个Agent意见、冲突仲裁、评分定级、生成结构化报告 | 大模型推理 + 多Agent协商机制 |

### 冲突仲裁示例
当**电气一次Agent**认定开关柜数量为5面，而**造价分析Agent**从清单中提取到6面时，主审Agent自动触发冲突仲裁：
1. 回溯图纸标注与清单来源
2. 以图纸标注为准，确认6面
3. 标记为"需人工复核"
4. 在报告中注明差异原因

## 四、融合后的关键技术实现

### 4.1 多智能体协作框架（LangGraph）
- **并行节点**：7个专业Agent同时运行，互不阻塞
- **汇聚节点**：主审Agent等待所有专业Agent输出后触发
- **条件边**：若任一Agent发现"严重缺陷"，立即触发人工介入节点中断流程
- **循环边**：设计师修改后可重新提交，触发增量评审

### 4.2 图纸理解（多模态大模型）

| 能力 | 实现方式 |
|---|---|
| 矢量化解析 | DWG→结构化拓扑数据（设备节点、连接关系） |
| 视觉理解 | Qwen-VL / GPT-4o 识别图签、标注、尺寸 |
| 跨图一致性 | 对比电气接线图与土建平面图的间隔编号、设备坐标 |
| 工程量提取 | OCR识别杆塔明细表、设备材料清册 |

### 4.3 规范知识库（RAG增强检索）
- **切片策略**：按"条文编号 + 条款内容 + 适用范围"组织chunk
- **混合检索**：向量语义检索（相近含义）+ 关键词过滤（限定电压等级、工程类型）
- **引用溯源**：Agent输出意见时必须附带规范条文编号及原文，确保可解释性
- **版本管理**：自动关联最新有效条文，标记废止替代关系

### 4.4 造价指标库（时序+对比分析）
- **特征工程**：项目类型、电压等级、建设年代、地区系数
- **异常检测**：孤立森林算法识别偏离同类项目±20%以上的单价
- **工程量联动校验**：图纸Agent识别的设备数量 vs 清单Agent提取的数量，自动比对一致性

### 4.5 领域大模型训练路径

| 阶段 | 周期 | 内容 | 数据量 |
|---|---|---|---|
| 规范预训练 | 2个月 | 投喂《电力工程电气设计手册》等千级标准 | 500万+ tokens |
| 评审指令微调 | 1个月 | "设计描述→规范条文→评审意见"指令对 | 10万+ 指令对 |
| RAG增强 | 持续 | 接入企业私有知识库（历史评审纪要、内部技术规定） | 动态更新 |

## 五、典型评审场景

### 场景1：输变电工程初步设计评审
**输入**：初设说明书、电气主接线图、总平面布置图、主要设备材料清册
**流程**：
1. 合规Agent扫描全文，标出与GB 50059等不符的条款
2. 电气一次Agent校验主变容量与系统规划负荷匹配度
3. 土建Agent评估站区竖向设计与防洪标准符合性
4. 主审Agent聚合意见，生成结构化报告（重大/一般/建议分级）

### 场景2：新能源接入系统评审
**输入**：光伏电站接入方案、电能质量评估报告
**流程**：
1. 电气一次Agent核查逆变器低电压穿越能力
2. 电气二次Agent评估调度自动化信息上传完整性
3. 安全评估Agent识别防孤岛保护配置缺失
4. 造价Agent检测组件单价异常偏离

### 场景3：施工图设计质量抽检
**输入**：施工图纸、计算书
**流程**：
1. 多模态模型解析图纸标注尺寸，与计算书交叉验证
2. 线路Agent自动核算导线弧垂对地距离
3. 发现"图纸标注12m，计算书显示11.2m（不满足规程）"类隐性错误
